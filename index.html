🧰 A Starting Kit

Everything in the Athena Framework is built to let you code your roleplay game mode faster. Skip writing the hard stuff, and focus on the cool stuff.

🛠️ Powered by TypeScript

Athena Framework uses modern development tools such as TypeScript, Vite, Vue 3, and MongoDB. This makes it so that familiar developers who have jobs in the industry are familiar with the tools that the Athena Framework uses.

👋 Come Join Us!

We love teaching new people how to program, and how to get involved in the project. Come build alongside established developers and learn a new hobby.
